# crx_3dme_rp
FiveM /me chat script for players to use

# Steps on installing
* Drag and drop crx_3dme_rp into your resource folder
* Add paste the following in your server.cfg
* ensure crx_3dme_rp
  
# chat command #
```
"/me ex: How many GSW's?"
```
* start the server
* ensure the resource has started in your console log
* run "/me" and boom your done
